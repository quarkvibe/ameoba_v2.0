## üèóÔ∏è Architecture Overview

Amoeba follows a **cellular architecture** pattern:

- **NUCLEUS** (Core) - Business logic
- **RIBOSOMES** (Routes) - HTTP request handling
- **GOLGI** (Services) - Processing & packaging
- **MITOCHONDRIA** (Database) - Data storage
- **CILIA** (Integrations) - External services

**Key Principle:** One service (blob), multiple access points (cilia)

**Example:** Testing system accessible via API, SMS, CLI, and Dashboard - all using one service.

See [ARCHITECTURE.md](../ARCHITECTURE.md) for complete details.
